COE Digital Assistant

The next-generation AI gateway for G.C. Sanjauli students to manage academic life with precision.

Project Overview

Designed specifically for the Centre of Excellence, Sanjauli, this assistant acts as a centralized node for student services. It utilizes advanced Natural Language Processing to bridge the gap between complex college databases and student queries.

Core Capabilities:

  • Smart Query Resolution: Immediate answers about college courses, faculty, and general info.
  • Live Academic Tracking: Direct access to attendance records and internal grades via MySQL.
  • Document Intelligence: Capability to process and retrieve information from uploaded PDFs and notices.
  • Dynamic Scheduling: Real-time timetable retrieval based on your specific course and semester.

Technology Stack

Llama 3.3 (70B) Groq Cloud API Python Flask MySQL Database RESTful Architecture PyPDF2 Intelligence

Security & Privacy

  • Session Integrity: Strict login protocols for Admin and Student accounts.
  • Data Isolation: Student profiles are logically separated in the database to prevent leaks.
  • Input Validation: All data (like Roll Numbers) undergoes server-side validation to prevent SQL injection.
  • QR Verification: Approved ID cards include unique hash identifiers for physical verification.
